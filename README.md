# Task 4: Real-Time Multi-Object Detection & Analytics

## InternGrow AI Track

### About

This project uses YOLOv8 to detect objects from a live webcam feed in real-time. It draws bounding boxes around detected objects with labels and confidence scores.

Upgrade Feature: An automated counting zone that tracks people, cars, buses, and trucks crossing a virtual line on the screen.

### Technologies

- Python 3.14
- YOLOv8 (Ultralytics)
- OpenCV
- NumPy

### How to Run

pip install opencv-python ultralytics numpy

python task4_basic.py

python task4_upgrade.py

Press 'q' to exit the program

### Files

task4_basic.py - Basic object detection
task4_upgrade.py - Detection + counting zone
